1.下载问题

由于网速较慢经常终止，再次重新下载时会遇到这种情况

[[16415b8f7ea1ded8983c27a13c49120f_MD5.jpeg|Open: 61b0e720ee5027b540d4196dfe2819b6_MD5.png]]
[[source/_posts/工具/Anaconda + Pytorch 安装教程/16415b8f7ea1ded8983c27a13c49120f_MD5.jpeg|Open: b387a0852b47d24cd351267a4c7131d4_MD5.jpeg]]
![[source/_posts/工具/Anaconda + Pytorch 安装教程/16415b8f7ea1ded8983c27a13c49120f_MD5.jpeg]]

错误原因:之前下载中断，导致索引关系发生错误

解决办法

在pytorch虚拟环境下输入命令

conda clean -i

[[8c17cd156a68f49551df167bc6e89fcc_MD5.jpeg|Open: 1320e4664ff05b29354b42072f60f509_MD5.png]]
[[source/_posts/工具/Anaconda + Pytorch 安装教程/8c17cd156a68f49551df167bc6e89fcc_MD5.jpeg|Open: 1c6cdadd1b11504374a583cebc083d0e_MD5.jpeg]]
![[source/_posts/工具/Anaconda + Pytorch 安装教程/8c17cd156a68f49551df167bc6e89fcc_MD5.jpeg]]
然后按y即可取消索引重新下载pytorch包
即重新键入

conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia
